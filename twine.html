Twine

What is Twine?

Twine is an open source program where you can create a website to tell a story. It requires no coding but incorporates coding aspects like variables and conditional logic.

To learn more, visit Twine site.




My Twine Game

My game takes inspiration from many of the RPG games I have played like Final Fantasy. It provides a deep lore that can be easily expanded in the future and an engaging story. The game itself is based on the importance of family. I grew up with a tightly knit family and was taught that family is one of the most important things to treasure if possible.
