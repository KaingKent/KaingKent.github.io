Projects

Here are some notable projects I have completed


osu!keep

  • Launched a website using the Google Cloud Platform and Flask to provide more user data for players of the online rhythm game osu! than the original website
  • Players are able to login to their osu! account through the osu! API or search another user’s name providing them their top 100 scores and their most recent scores
  • Users are able to sort the tables by different categories and comment on other players’ profiles to give feedback

Search Engine

  • Developed an application where the user is able to search for a term or find the top-n most used term that runs on Docker
  • Application uses GCP clusters to run Apache Hadoop jobs that use mapreduce to build inverted indices and find the top-n

Billiards Animation

  • Created a world with sixteen billiard balls, each with their own texture, and accurate, moveable lighting and shadows
  • Users are able to move the light bulb, camera and can start a series of animations using the keyboard
